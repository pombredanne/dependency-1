Generated Code
==============
The .apidoc file lists the specific targets which are generated by
apidoc. To regenerate code, clone github.com/gilt/apidoc-cli and

    apidoc upload bryzek dependency ./spec/dependency.json --version 0.0.2-dev
    
    apidoc update
